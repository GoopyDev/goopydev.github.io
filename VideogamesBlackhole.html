Videogames Blackhole

Videogames Blackhole is a comprehensive database of video games powered by the RAWG API, boasting information on over 900,000 titles. The user-friendly interface offers sorting filters based on alphabetical order or ascending/descending ranking (ratings). Each game is tagged with relevant categories, allowing users to filter their searches accordingly.

Users can search for games by name (employing a broad search that retrieves all games containing the specified string) or by the unique ID number assigned to each game in the library (exact search). Additionally, the platform provides the capability to create new games within a user's personal database, utilizing a controlled form that is validated by both the frontend and server to ensure consistent information structure and completeness.

The technologies utilized include JavaScript, Node.js, React, Redux, PostgreSQL, StyledComponents, Express, and Sequelize.
