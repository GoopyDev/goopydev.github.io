Projects

In this section, I've compiled a collection of projects that have been instrumental in shaping my skills as a developer and game enthusiast

ElectroShop

An online gaming store where clients can browse gaming products, add items to a shopping cart, create an account and make purchases using MercadoPagoSDK.

An admin dashboard is available to create and upload images for new products.

The Sacred Fork

Discover a gastronomy site powered by Spoonacular API, allowing users to easily search food recipes by name, keywords, or ID. Filter options by preferred diets and sort results by name or health score. Users can create their own recipes by specifying the steps to follow and adding a dish image through a controlled form.

Videogames Blackhole

Dive into the videogames universe to find up to 900.000 titles. This gaming website is fed by the powerful RAWG API, and provides game search, filtering, detailed info, and a form to create your own game data.

Special Girl

A 3D game where players control a character by clicking on the environment using navmesh technology. The game is set in a diverse landscape of mountains, lush green areas, and trees. However, the adventure begins within the confines of a building resembling an abandoned hospital.

The Duke's Order

An immersive 2.5D platformer where players embody a sword-wielding knight on a mission to vanquish a menacing dragon and save the kingdom. The game combines 3D environments with 2D movement mechanics, showcasing intricate character animation powered by Unity's "Anima2D" technology. Players engage in sword combat, with three distinct attack animations triggered by multiple button presses. It's a technically enriched 2.5D adventure featuring captivating combat and dynamic animations.

La tanqueta

This game offers an engaging 3D gaming experience designed for two players in split-screen. Maneuver unique tanks with independent turret aiming, capturing the flag or eliminating opponents in dynamic environments with cover. The game features diverse power-ups and challenges for an exciting tank duel adventure.
